# AI 算力与大模型行业财务分析报告

> **项目状态**：第一阶段（战略框架）已完成，第二阶段（财务量化建模）进行中  
> **更新预期**：2026年Q3 完成杜邦分析、盈亏平衡模型及供应链风险量化

## 📌 项目定位

本项目从 **财务视角** 系统分析中国 AI 算力与大模型行业，区别于市面常见的战略/技术报告。  
目标读者：投资机构分析师、企业财务总监、以及关注 AI 商业化的产品经理。

**核心问题**：
- 为什么 AI 企业营收翻倍却普遍亏损？亏损结构来自哪里？
- 盈利拐点何时出现？关键敏感因子是什么？
- 地缘政治如何映射为利润表上的数字？
